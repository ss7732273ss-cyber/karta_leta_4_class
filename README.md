# 🗺️ Читательский дневник-экспедиция: Карта летних приключений после 4 класса

Интерактивный, красочный читательский дневник для детей, закончивших 4 класс. Позволяет отмечать прочитанные книги, заполнять отчеты в игровой форме (выбор любимых цитат, оценка волшебности, выбор действий за главного героя) и отслеживать прогресс покорения «вершин» на интерактивной карте летней экспедиции.

## 🚀 Особенности проекта

- 🧭 **Интерактивная карта**: Каждое литературное произведение — это вершина или локация, которую юный читатель посещает и отмечает на карте.
- 🎒 **Игровой формат отчетов**: Вместо скучных пересказов ребёнок выбирает, как поступил бы на месте героя, отмечает самые забавные или волшебные теги.
- 🎨 **Потрясающий летний дизайн**: Светлая песочно-солнечная тема, плавная анимация движения по точкам карты, поддержка детских аватаров и настраиваемого имени.
- 🏅 **Награды и ачивки**: Получение красочных значков за прочитанные книги разной тематики (Долина преданий, Берега природы, Перевал мужества, Горизонты открытий).
- 🖨️ **Печать дневника**: Специально разработанный формат печатной версии, позволяющий распечатать дневник или красивую сводную таблицу в один клик.

## ⚙️ Технологии

- **React 18** + **TypeScript**
- **Vite** для быстрой сборки
- **Tailwind CSS** для стильного адаптивного интерфейса
- **Motion** (`motion/react`) для интерактивной анимации
- **Lucide React** для иконок

## 📦 Сборка и Деплой

Проект настроен для автоматической сборки и публикации на **GitHub Pages** с помощью GitHub Actions.

### Локальная разработка:
```bash
npm install
npm run dev
```

### Сборка проекта:
```bash
npm run build
```
Собранные файлы сохраняются в папку `dist/`.

### Автоматический деплой:
Для публикации на GitHub Pages:
1. Перейдите в настройки вашего репозитория на GitHub: **Settings -> Pages**.
2. В разделе **Build and deployment** выберите **Source: GitHub Actions**.
3. После этого при каждом пуше в ветку `main` или `master` проект будет автоматически собираться и публиковаться из скомпилированной папки `dist/`.
